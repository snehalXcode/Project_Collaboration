<h1 align="center">🚀 Smart Group Formation</h1>
<h3 align="center">AI-Assisted Student Team Creation Platform</h3>

---

## 🌐 Overview

**Smart Group Formation** is an AI-assisted web platform that simplifies the process of forming academic project teams.

Instead of manually assigning students to groups, the system allows users to:

- Select subjects  
- Create collaborative teams  
- Manage project groups through a centralized dashboard  

The platform is designed for academic institutions where multiple courses require collaborative projects.

---

## ✨ Features

- 🔐 Secure user login system  
- 📊 Interactive dashboard displaying subjects  
- 👥 Create project teams dynamically  
- 🧠 AI-assisted collaboration planning  
- 📁 Centralized student & group management  
- 🎨 Clean and responsive UI  

---

## 🧭 Dashboard Modules

The dashboard contains interactive subject cards where students can create and manage groups.

### Supported Subjects

- 🧠 Artificial Intelligence  
- ☁️ Cloud Computing  
- 🌐 Computer Networks  
- 🗄 DBMS  
- 📊 Data Structures  
- 🤖 Machine Learning  
- 💻 Operating Systems  
- ⚙ Software Engineering  
- 🌍 Web Technologies  

Each subject module allows students to create or manage project teams.
